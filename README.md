## Installation
1. **Clone the repository:**
    ```bash
    git clone https://github.com/Ama4538/Etch-A-Sketch.git
    cd PortfolioV3
    ```
2. **Install dependencies:**
    ```bash
    npm i
    ```
3. **Build and Start the server:**
    ```bash
    npm build
    npm start
    ```
4. **Access the application:**
Open your browser and go to localhost
